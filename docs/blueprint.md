# **App Name**: SOON Website

## Core Features:

- Landing Page: Single-page marketing website with sections for hero, services, features, coverage, pricing, FAQ, and contact.
- Admin Dashboard: Login-protected dashboard with content management capabilities.
- Contact Form: Contact form with validation that uses a Next.js Server Action to submit data and display success/error messages.
- Content Management: Ability to edit website content (text, images, etc.) via the admin dashboard. Can use tools like markdown editor or WYSIWYG editor.

## Style Guidelines:

- Primary color: Deep, authoritative red (#B32525) for CTAs and highlights.
- Muted Background: Light grays (#F9FAFB and #F0F1F3) for alternating section backgrounds.
- Background color: Clean, near-white color (#F2F3F5).
- Text/Foreground: Very dark, almost black color (#0A0E1B) for maximum readability.
- Admin Sidebar: Dark navy/blue theme (#141E33) for the background with light foreground text.
- Body font: 'Inter' sans-serif for a modern, readable feel.
- Headline font: 'Playfair Display' serif for elegant headings. Note: currently only Google Fonts are supported.
- Fully responsive layout using Tailwind CSS, ensuring optimal viewing experience across devices.
- Subtle fade-in-on-scroll animations using react-awesome-reveal for major sections.
